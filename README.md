## Installation

[Available in Hex](https://hex.pm/packages/hog), the package can be installed by adding `hog` to your list of
dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:hog, "~> 0.1.0"}
  ]
end
```

Documentation can be found at [https://hexdocs.pm/hog](https://hexdocs.pm/hog).
